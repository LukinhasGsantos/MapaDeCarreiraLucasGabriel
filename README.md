# Meu Mapa de Carreira

Rascunho inicial de uma página estática para mapa profissional.

## Arquivos principais

- `index.html`: conteúdo do mapa de carreira.
- `styles/style.css`: aparência visual e responsividade.
- `mapadecarreira.js`: animações de entrada e barras de habilidades.
- `assets/images/profile-placeholder.svg`: imagem provisória de perfil.

Para revisar no navegador, abra o arquivo `index.html`.

Se preferir abrir por servidor local, execute:

```bash
node preview-server.cjs
```

Depois acesse `http://127.0.0.1:5177`.
